# 问题三：基于质量守恒与 GRU 的混合动态模型构建

## 1 问题背景

出厂水浊度（NTU）是衡量供水水质的重要指标，也是水厂运行管理的核心目标之一。

然而，从原水进入系统到最终成为出厂水，需要经历混凝、沉淀、过滤以及清水池调蓄等多个工艺环节，因此出厂水浊度不仅受当前时刻运行状态影响，还受到历史工况以及水力停留时间（Hydraulic Retention Time, HRT）的影响。

由于系统具有明显的时滞特征，直接建立未来 1–12 小时出厂水浊度预测模型具有较大难度。

题目要求结合：

* 质量守恒原理；
* 水力停留时间分布；
* 数据驱动方法（GRU、LSTM 或状态空间模型）；

构建混合动态预测模型，实现未来出厂水浊度预测。

---

# 2 建模思路

本研究采用：

> 质量守恒机理模型 + GRU 神经网络

构建混合动态预测框架。

其中：

* 质量守恒模型负责刻画清水池中的物质传输规律；
* GRU负责学习复杂非线性关系以及时间滞后效应；
* 两者结合形成具有物理解释性的动态预测模型。

整体流程如下：

```text
原始运行数据
      ↓
质量守恒公式分解
      ↓
构造物理特征
      ↓
时间序列样本构建
      ↓
GRU网络训练
      ↓
预测未来NTU增量
      ↓
恢复未来NTU
      ↓
输出未来2~12小时预测结果
```

---

# 3 质量守恒机理分析

## 3.1 基本质量守恒思想

将清水池视为控制体，则池内浊度变化可表示为：

[
\frac{d(VC)}{dt}
================

## Q_{in}C_{in}

Q_{out}C
]

其中：

[
C
]

表示当前出厂水浊度；

[
C_{in}
]

表示进入清水池的来水浊度；

[
Q_{in}
]

表示进水流量；

[
Q_{out}
]

表示出水流量；

[
V
]

表示池体有效体积。

---

## 3.2 离散化表达

由于数据采样间隔为 2 小时，因此采用离散形式：

[
C_{t+1}
=======

C_t
+
\frac{\Delta t Q_{in,t}C_{in,t}}{V_t}
-------------------------------------

\frac{\Delta t Q_{out,t}C_t}{V_t}
]

其中：

[
\Delta t = 2h
]

表示相邻采样时刻之间的时间间隔。

该表达式反映：

* 入流水质会提高池内浊度；
* 出流水会带走池内污染物；
* 池体容量具有缓冲和调蓄作用。

---

# 4 物理特征构建

为了使神经网络能够学习物理规律，将质量守恒公式拆分为多个独立特征输入。

---

## 4.1 当前状态项

当前出厂水浊度：

[
S_t=C_t
]

对应特征：

```text
current_NTU
```

表示系统当前状态。

---

## 4.2 入流项

根据质量守恒公式构造：

[
I_t
===

\frac{2Q_{in,t}C_{in,t}}
{L_t}
]

其中：

* (L_t) 为清水池水位；
* 作为池体有效体积代理变量。

本研究分别测试两种方案：

### 方案A

采用原水流量：

[
Q_{in}=R/W\ FLOW
]

即：

[
I_t^{RW}
========

\frac
{2(R/W\ FLOW)\times(FILT.NTU)}
{C/W\ WELL\ LEVEL}
]

---

### 方案B

采用出厂流量：

[
Q_{in}=T/W\ FLOW
]

即：

[
I_t^{TW}
========

\frac
{2(T/W\ FLOW)\times(FILT.NTU)}
{C/W\ WELL\ LEVEL}
]

用于分析不同流量代理对预测性能的影响。

---

## 4.3 出流项

构造：

[
O_t
===

\frac
{2(T/W\ FLOW)\times NTU}
{C/W\ WELL\ LEVEL}
]

对应质量守恒中的流出部分。

该项反映系统当前污染物输出强度。

---

## 4.4 水力停留时间代理

题目特别强调：

> 清水池水力停留时间分布

因此构造：

[
HRT_t
=====

\frac
{C/W\ WELL\ LEVEL}
{T/W\ FLOW+\varepsilon}
]

作为水力停留时间代理变量。

其物理意义为：

* 水位越高；
* 出厂流量越小；

则水体在池内停留时间越长。

---

## 4.5 水位变化特征

构造：

[
\Delta L_t
==========

L_t-L_{t-1}
]

对应：

```text
WELL_LEVEL_CHANGE
```

用于描述池体储量变化过程。

---

# 5 GRU 时间序列建模

## 5.1 输入窗口

数据采样周期：

[
2h
]

采用：

[
lookback=12
]

即：

[
24h
]

历史窗口。

因此模型输入为：

[
X_t=
[x_{t-11},x_{t-10},...,x_t]
]

对应过去一天运行状态。

---

## 5.2 输入特征

每个时间点包含：

| 特征                | 说明     |
| ----------------- | ------ |
| current_NTU       | 当前出厂浊度 |
| inflow_term       | 入流项    |
| outflow_term      | 出流项    |
| HRT_PROXY         | 水力停留时间 |
| WELL_LEVEL_CHANGE | 水位变化   |
| FILT. NTU         | 滤后水浊度  |
| R/W FLOW          | 原水流量   |
| T/W FLOW          | 出厂流量   |

共：

[
8
]

个输入变量。

因此输入张量为：

[
[batch,12,8]
]

---

## 5.3 输出设计

直接预测未来 NTU 容易受到强自相关影响。

因此采用残差预测：

[
\Delta C_{t+h}
==============

C_{t+h}-C_t
]

预测目标：

[
[\Delta C_{t+1},...,\Delta C_{t+6}]
]

对应：

* 2小时；
* 4小时；
* 6小时；
* 8小时；
* 10小时；
* 12小时。

最终恢复：

[
\hat C_{t+h}
============

C_t+\widehat{\Delta C}_{t+h}
]

得到未来实际 NTU。

---

# 6 GRU 网络结构

采用单层 GRU：

```text
Input
 ↓
GRU(64)
 ↓
Dropout
 ↓
Linear
 ↓
6-step output
```

数学形式：

[
h_t
===

GRU(x_t,h_{t-1})
]

其中：

[
h_t
]

表示历史24小时信息压缩后的隐藏状态。

随后：

[
\widehat{\Delta C}
==================

W_oh_t+b_o
]

输出未来六步增量预测。

---

# 7 训练策略

## 7.1 数据划分

采用时间顺序划分：

| 数据集        | 比例  |
| ---------- | --- |
| Train      | 70% |
| Validation | 15% |
| Test       | 15% |

避免时间泄漏。

---

## 7.2 标准化

仅使用训练集统计量：

[
z
=

\frac{x-\mu}{\sigma}
]

验证集与测试集使用相同参数。

---

## 7.3 损失函数

采用 Huber Loss：

[
L_\delta(e)
===========

\begin{cases}
\frac12 e^2,
&
|e|\le\delta
\
\delta(|e|-\frac12\delta),
&
|e|>\delta
\end{cases}
]

相比 MSE：

* 对异常峰值更稳定；
* 不易被少数极端样本主导。

---

## 7.4 Early Stopping

监控验证集损失：

```text
Validation Loss
```

若连续若干 Epoch 无改善：

```text
patience = 10~15
```

则提前停止训练。

防止模型出现过拟合。

# 9 模型物理意义

本模型并非纯数据驱动模型。

质量守恒部分提供：

* 入流影响；
* 出流影响；
* 清水池调蓄作用；
* 水力停留时间效应；

GRU部分负责学习：

* 时间滞后关系；
* 非线性变化规律；
* 复杂运行工况影响。

因此该模型同时具有：

* 物理可解释性；
* 动态预测能力；
* 多步预测能力。

符合题目要求的“质量守恒 + 数据驱动”混合动态建模思想。

# 10 结论

本文构建了一种基于质量守恒原理与 GRU 神经网络的混合动态模型。

通过将质量守恒公式拆解为入流项、出流项、水力停留时间及储量变化等物理特征，再结合 GRU 对时间滞后和非线性动态关系的学习能力，实现了未来 2–12 小时出厂水浊度的多步预测。

该方法兼顾物理机理解释与数据驱动建模优势，能够有效刻画供水系统中的动态传输过程，为后续水质预测与运行优化提供可靠依据。
